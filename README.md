# Lambeth Recycle Banks

A simple demonstration of ingesting, transforming and visualising geolocation data.

## Inputs

* [Source](https://lambethopenmappingdata-lambethcouncil.opendata.arcgis.com/datasets/recycling-banks)
* [GeoJSON File](dat/Recycling_Banks.geojson)

### Tools

* [Python](https://www.python.org/)
* [MySQL](https://www.mysql.com/)
* [Postcodes.io](https://postcodes.io/)

### Outputs

* [CSV file](out/geo_recycle_banks.csv)
* [Google Sheet](https://docs.google.com/spreadsheets/d/1H-BDiiq03LDyta0E-4HNK023c2S_pwMtaMXNDDb7a6M/edit?usp=sharing)
* [Example dashboard with heatmap and filters, with data from a Google Sheet](https://lookerstudio.google.com/reporting/d5c4c3fd-e60c-4085-be4b-ccb21d15914c/page/vE6vD)
